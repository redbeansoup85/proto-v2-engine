# v1.0 Manifest

## Guaranteed Capabilities
- Judgment-only architecture
- Explicit Responsibility Gate (Gate 1)
- Explicit Execution Authorization Gate (Gate 2)
- Artifact-only delivery (no auto execution)
- Full audit trail (receipt → plan → queue → processed)

## Explicitly Out of Scope
- Autonomous execution
- Self-triggered actions
- Implicit responsibility
- Hidden side effects
