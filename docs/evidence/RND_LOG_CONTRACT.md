# R&D Evidence Log Contract (v1.0)

## Rule
- Frequency: weekly (or milestone-based)
- Format must include:
  1) Technical Uncertainty
  2) Activities (what changed)
  3) Outputs (files/PRs)
  4) Verification (tests/CI)
  5) Time basis (reasonable estimate)

## Evidence Sources
- Git commits / PRs
- CI logs (GitHub Actions)
- audits/* hash-chain logs
- replay reports
